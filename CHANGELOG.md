# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-07

### Added
- **Initial Soft Release**
- Core Plugin Architecture with Material 3 Fluent design.
- **Plugins:**
  - Timer & Countdown: Stopwatch, countdown, and Unix converter.
  - Data Generator: UUID, Identity, and Text generation.
  - Clipboard Manager: History tracking and management.
  - Color Picker: Screen-wide color sampling.
  - Screen Recorder: High-performance recording with floating controls.
  - Screenshot: Full-screen and region capture.
  - Magic 8-Ball: QA decision-making tool.
  - Security Payloads: Common test strings for XSS and SQLi.
  - Settings: Theme management and licensing dashboard.
- GitHub Actions CI/CD for automated Windows releases.
